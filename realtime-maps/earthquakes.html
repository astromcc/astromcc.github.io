World-wide Earthquakes for the Past 30 Days

  • Data are retrieved from the live USGS archive of earthquake data for the past 30 days.
  • The USGS website Latest Earthquakes is a clickable, zoomable map with a sortable list of individual earthquakes.

This page makes use of p5js, largely based on Coding Challenge #57 of Dan Shiffman's Coding Train. The background world map is generated by the Static Image API from Mapbox.
