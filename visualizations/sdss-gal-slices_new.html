Mapping Galaxies in the Sloan Digital Sky Survey

  • These data come from Data Release 16 of the Sloan Digital Sky Survey (SDSS webpage for details).
  • Galaxies are selected based on a minimum brightness. Magnitude limits per filter: u < 22.0, g < 22.2, r < 22.2, i < 21.3, z < 20.5
  • Each image below displays a selection of galaxies with a range of redshifts. Each blue dot is a single galaxy.
  • Each range is selected to display a different distance from us and a different time in cosmic history.
  • Clicking each button on the left-hand navigation bar below selects a different redshift range.
