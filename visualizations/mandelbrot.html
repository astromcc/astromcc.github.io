The Mandelbrot Set

  • This is a visualization of the Mandelbrot set based on Coding Challenge #21 from the YouTube channel The Coding Train.
  • The Mandelbrot set was first defined in 1978. But it was first seen by Benoit Mandelbrot in 1980 during his work with computer visualization IBM. The set is constructed in the complex plane, the 2-d plane defined by real and imaginary numbers.
  • The graph shows different regions which are colored according to a simple equation, z(n+1) = z(n)^2 + c. The values of z are complex, having a form of a + bi, where i^2 = -1. Real values are plotted along the horizontal axis and imaginary values are plotted along the vertical axis.
